 Lab 4: Data Preprocessing

This repository contains the tasks completed for Lab 4, focusing on data cleaning and preprocessing using the House Prices dataset.

Tasks Performed:
- Task 1:Identified data quality issues and checked for missing values.
- Task 2:Handled missing values using the Median strategy to ensure robustness against outliers.
- Task 3:Detected and removed outliers using the IQR (Interquartile Range) method.
- Task 4: Normalized numerical features using both Min-Max Scaling** and Z-score Standardization.
- Task 5:Applied PCA (Principal Component Analysis) on correlated features to reduce dimensionality.

Files:
- `Lab4.ipynb`: The Jupyter Notebook containing the code and results.
- `train.csv`: The dataset used for the assignment.
